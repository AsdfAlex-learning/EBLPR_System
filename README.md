# 电动自行车车牌识别系统

## 环境配置

### Windows环境配置步骤

1. 安装依赖
   ```powershell
   # 以管理员身份运行PowerShell
   cd license_plate_recognition
   .\scripts\setup_venv.ps1
   ```

2. 激活虚拟环境
   ```powershell
   .\venv\Scripts\Activate.ps1
   ```

3. 验证安装
   ```powershell
   python -c "import fastapi; print('环境配置成功')"
   ```
